DocPortal — Sistema de Documentación Interna
=============================================

Bienvenido al portal de documentación corporativa de InfraCorp S.L.

Este sistema centraliza todos los manuales, guías y registros de cambios
del departamento de operaciones. El acceso está restringido al personal
con credenciales activas.

Para soporte técnico, contacta con el equipo de IT en it-support@infracorp.internal.
